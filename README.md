# 深淵巢穴 (Deep Abyss Hive)

## 項目概述

深淵巢穴是一款創新的 Idle RTS 遊戲，玩家扮演深淵母巢的集合意志，通過戰略決策指導蟲群的無限擴張與進化。

## 核心特色

- **Idle RTS**: 戰略決策 + 自動執行
- **母巢意志**: God視角的統御體驗  
- **無限進化**: 深度的生物進化系統
- **宇宙征服**: 跨維度的探索擴張

## 技術架構

- **客戶端**: Unity 2022.3 LTS
- **服務端**: Node.js + TypeScript
- **數據庫**: MongoDB + Redis
- **部署**: Docker + Kubernetes
